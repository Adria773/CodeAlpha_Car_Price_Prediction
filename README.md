# CodeAlpha_Car_Price_Prediction
# Car Price Prediction using Machine Learning 🚗

## Project Overview
This project predicts the selling price of used cars using machine learning algorithms.

## Dataset
The dataset contains various features related to cars such as:
- Car Name
- Year
- Present Price
- Fuel Type
- Transmission
- Owner
- Selling Price

Dataset Source:
https://www.kaggle.com/datasets/vijayaadithyanvg/car-price-predictionused-cars

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Model
Linear Regression was used to predict the car prices.

## Project Steps
1. Data preprocessing
2. Feature selection
3. Model training
4. Model evaluation

## Result
The model predicts approximate selling prices based on car features.
